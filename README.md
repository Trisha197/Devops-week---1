# DevOps CI/CD Pipeline Demo 🚀

A simple website demonstrating our DevOps CI/CD pipeline with GitHub Actions and Vercel.

## 🎯 What This Demonstrates

- **GitHub Actions**: Automated deployment on push to main branch
- **Vercel**: Instant deployment and hosting
- **CI/CD Pipeline**: Code → Build → Test → Deploy flow
- **DevOps Culture**: Infrastructure as Code, Automation, Monitoring

## 🛠️ Tech Stack

- HTML5 + CSS3 + JavaScript
- GitHub Actions (CI/CD)
- Vercel (Hosting)
- Git (Version Control)

## 🚀 How to Deploy

1. Fork this repository
2. Create a Vercel account
3. Import this repository on Vercel
4. Add Vercel secrets to GitHub:
   - `VERCEL_TOKEN`
   - `VERCEL_ORG_ID`
   - `VERCEL_PROJECT_ID`
5. Push to main branch → Auto-deploys!
